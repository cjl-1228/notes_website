 為甚麼許多軟體工程師寫程式都選擇Linux大於Windows的原因


The open-source operating system, Linux is often the default choice for developers. The OS offers powerful features to developers. The Unix-like system is open to customisation, allowing developers to change the OS as per there needs.

Why developers choose Linux over Windows

The primary reason behind the popularity of Linux is the availability of compilers or interpreter. You don't have to download them yourself. On the other hand, Windows does not come with pre-installed compilers. If Linux does not have what you need, you can still get all the functionalities that you need from the command line. Simply enter the command and it will atomically download and install the package. Which saves a lot of efforts.

Linux contains low-level tools like sed, grep, awk piping etc. Programmers working on complex applications love it for Linux's versatility, power, security, and speed.

1. GNU software with Linux kernel

Linux OS consists of a combination of GNU software with a Linux kernel. It is important to understand that GNU is a replacement for Unix. Linux does not contain any Unix-based code. On the other hand, Linux contains an extensive collection of software. The project is shipped with Kernel and GNU HURD which is a multi-server microkernel written as part of GNU.

2. Support for programming languages

Linux supports almost all the programming languages including the popular ones like C, C++, CSS, PHP, Java, JavaScript, HTML, Perl, Python, Ruby, or Vaya. Any programming language that is not limited to a specific OS like Visual Basic, it will work on Linux. If you run into support issues, you can always find the required package from Linux's distribution repositories.

3. Choice of pre-installed apps

The OS has a lot of apps to choose from. You can just write your code using a simple text file and then run the same in Linux. By default, you get all necessary apps like Gedit, Kate, Emacs, nano, and Vim.
